Começando pelo fundamental: HTML (HyperText Markup Language)

  • Você sabe o significado da sigla HTML? É Linguagem de Marcação de Hypertexto!
  • Lembre-se sempre que HTML não é uma linguagem de programação, e sim de marcação.
  • O HTML é mais importante do que muitas pessoas pensam. Ele é responsável por fazer toda a parte estrutural do web site, sem ele, não existiria uma forma de escrever um website!

Agora que você já sabe o que significa o HTML, podemos começar a ver as tags usadas e como elas funcionam!

Vamos começar com Títulos e Paragráfos:

As tags utilizadas para títulos são de h1 até o h6 e funcionam em ordem decrecente, sendo h1 a maior fonte de títulos e h6 a menor. É importante você ter somente uma tag h1 no seu website, pois ela é usada para definir o título principal, as outras tags são usadas como subtítulo. Você pode obsevar isso nesse mesmo site, o título principal "Um guia de estudos sobre HTML e CSS" foi feito com a tag h1, enquando o "Começando pelo fundamental: HTML (HyperText Markup Language)" foi feito com o h2, o terceiro foi feito com o h3 e o outro com o h4!

Já o parágrafo tem uma única tag e é utilizada para textos comuns!

Como você pode ver, no texto assima foi utilizado a tag p, assim como nesse texto que você está lendo nesse exato momento. É importante que você saiba como como utilizar essas tags para deixar o seu website visualemnte organizado e bem trabalhado.

Okay, podemos ir para o próximo? Vamos falar sobre listas!

Listas são bem simples de se usar, e vou tentar explicar de uma forma bem prática.


Existem dois tipos de listas: A ordernada e a não ordernada."Tá, mas qual a diferença?" vocês me perguntam... A lista ordernada é formada numericamente, o primeiro item da lista será 1, o segundo será 2 e assim por diante. Já a não ordernada é formada pelo caractere "disc".

  1. Este é o primeiro item da lista
  2. Este é o segundo item da lista
  3. Este é o terceiro
  4. Deu pra entender?
  • Este é o primeiro item da lista
  • Este é o segundo item da lista
  • Este é o terceiro
  • Acho que deu sim

Para criar uma lista ordernada usamos a tag ol, para cada item que colocarmos, vamos usar a tag li dentro da tag ol. Confuso, né? Da uma olhadinha pra entender melhor:

A tag ol é a que define a lista ordernada, a tag li é a que faz cada item ser uma lista.

E assim funciona com a lista não ordernada, mas vamos utilizar a tag ul.

A tag ul é a que define a lista não ordernada, a tag li é a que faz cada item ser uma lista.

Agora vamos para um assunto mais complicado, links!

Usar links não é algo complexo, mas é importante você prestar bastante atenção nesse tópico.
Link são usados com a tag ancôra (a) e um atributo chamado href. Atributos são como uma palavra chave, ele é usado para vincular informações adicionais na tag, mas vamos falar mais afundo sobre isso depois! Voltando ao assunto, o atributo href é utilizado para adicionar um link que você deseja que ao clicar, o usuário seja direcionado.
